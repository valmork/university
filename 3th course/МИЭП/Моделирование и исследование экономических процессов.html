Моделирование и исследование экономических процессов

Моделирование и исследование экономических процессов

1. Моделирование как метод научного познания

Модель — это реально существующий или мысленно представляемый объект, который в процессе исследования заменяет объект-оригинал так, что его изучение даёт новые сведения об объекте-оригинале.

Моделирование — процес построения, изучения и примененеия моделей.

2. Экономико-математические методы и модели

Математические модели, наряду с базами данных, техническими средствами — важнейший компонент системы поддержки принятия решений.

Система поддержки принятия решений

система, которая позволяет использовать данные и модели для решения как слабо, так и хорошо структурированных проблем.

Классификация:

  1. По целевому назначению

    1. Теоретико-аналитические — для исследования наиболее общих свойств и закономерностей развития экономических процессов

    2. Прикладные — для решения конкретных задач

  2. По способу отражения фактора времени

    1. Динамическая

    2. Статическая

  3. По уровню экономических процессов

    1. Производственно-технологический уровень

    2. Социально-экономический уровень

  4. По характеру отражения причинно-следственных связей

    1. Детерминированные

    2. Стахостические

  5. По форме математических зависимостей

    1. Линейные

    2. Нелинейные

  6. По степени детализации

    1. Агрегированне (обобщённые)

    2. Детализированые

3. Принципы построения

  • Принцип достаточности исходной ифномации — используется только та информация, которая известна с необходимой точностью.

  • Принцип однозначности информации — информация не зависит от параметров моделируемой системы

  • Принцип преемствености — последующая модель не нарушает свойства объекта, отражённые в предыдущих

  • Принцип эффективной реализуемости — модель должна быть реализуема

4. Этапы экономико-математического моделирования

  1. Постановка задачи — постановка проблемы и её качественный анализ — необходимо сформулировать сущность проблемы, принимаемые допущения, а также вопросы, на которые требуется получить ответ. Определяются основные свойства моделируемого объекта и зависимости, связывающие его элементы. Формулируются гипотезы, предварительно объясняющее поведение объекта.

  2. Построение математической модели — выражение её в виде математических зависимостей и отношений. Излишняя сложность модели затрудняет процесс исследования и увиличивает затраты

  3. Математический анализ модели — определение её общих свойств и характеристик. Наиболее важно доказать существование решения в сформулированой модели.

  4. Подготовка исходной информации — информация в значительной степени определяет возможность построения конкретной модели.

  5. Численное решение — строятся алгоритмы, выполняются рассчёты.

  6. Анализ результатов и их применение.

На любом этапе возможен возврат на любой из предыдущих этапов.

Общая схема

В первом разделе n + 1 строка и n + 1 столбец. xij — то, сколько i-я отрасль продала j-й отрасли за год.

В i-й строке описано распределение продукции i-й отрасли как средства производства для других отраслей. Сумма \$sum_(j=1)^n x_(ij)\$ (строки) показывает производственное потребление продукции данной отрасли другими отраслями. В j-м столбце — потребление j-й отраслю продукции других отраслей. Сумма \$sum_(i=1)^n x_(ij)\$ (столбца) показывает производственные затраты отрасли.

Первый раздел показывает затраты и распределение продукции на производственные цели. Сумма \$sum_(i=1)^n sum_(j=1)^n x_(ij)\$ показывает промежуточный продукт экономики — сумму всех затрат отраслей.

Второй раздел посвящён конечному продукту y и валовому выпуску X. \$X_i = sum_(j=1)^n x_(ij) + y_1\$.

Конечный продукт yi — потребление, не идущее на текущие производственные нужды:

  • накопление и возмедение выбытия основных фондов (производственного оборудования)

  • прирост запасов

  • личное потребление населения

  • расходы государства

Конечный продукт входит в сальдо экспорта и импорта продукции.

Третий раздел отразажет стоимостную структуру валового продукта отраслей. \$V_i = X_j - sum_(i=1)^n x_(ij)\$.

Условно чистая продукция Vi делится на

  • арматизационные отчисления, идущие на возмещение выбытий основных фондов

  • чистую продукцию (зарплата, прибыль, налоги)

Четвёртый раздел находится под 2-м и отражает конечное распределение и использование национального дохода (∑ yi) в результате перераспределения. В разделе отражаются источники финансирования капиталовложений и текущие затраты непроизводственного сектора для анализа общей структуры конечных доходов по группам потребителей.

Связь между конечной и условно чистой продукцикй

\$X_i = sum_(j=1)^n x_(ij) + y_i\$ — основное уравнение баланса

\$X_j = sum_(i=1)^n x_(ij) + V_i\$

\$sum_(i=1)^n (sum_(j=1)^n x_(ij) + y_i) = sum_(j=1)^n (sum_(i=1)^n x_(ij) + V_i) => sum_(i=1)^n y_i = sum_(j=1)^n V_j\$

То есть, в 3-м разделе также фигурирует конечный продукт. Во 2-м разделе отражена структура потребления конечного продукта, а в 3-м — отражается, где произведена стоимость конечного продукта.

∑ V = ∑ y

y — стркутура конечного продукта

V — источники финансированяи конечного продуктк

Статическая межотраслевая модель

Коэффициент прямых материальных затрат

сколько единиц продукции i-й отрасли идёт на производство одной единицы производства j-й отрасли, то есть доля.

\$a_(ij) = x_(ij)/X_j\$

Допускается, что в каждой отрасли

  • выпускается единственный продукт по единственной технологии производства;

  • нормы затрат не зависят от выпуска продукции;

  • не допускается замещение одного вида сырья другим.

Тогда \$x_(ij) = a_(ij) X_j\$ или \$X_i = sum_(j=1)^n (a_(ij) X_j) + y_i\$

В матричном виде: X = AX + Y, где X = (x1, x2, …​, xn), Y = (y1, y2, …​, yn), A = {aij}nn (матрица).

aij могут быть получены

-статистически — на основе данных о прошлых периодах - нормативно — установив нормы затрат

Коэффициент полных материальных затрат

  • X = AX + Y

  • (I - A) X = Y

  • X = (I - A)-1 Y

Если (I - A)-1 существует и её элементы неотрицательны, то A — продуктивная матрица, то есть если существует неотрицательный вектор X⁰ > AX⁰

Свойства коэффициента прямых материальных затрат:

  1. Неотрицательность

  2. Сумма коэффициентов по столбцу меньше 1: \$sum_(i=1)^n a_(ij) < 1 <=> X_j > sum_(i=1)^n x_(ij) <=> X_j > X_j > sum_(i=1)^n (a_(ij) X_j)\$

Поскольку эти свойства выполняются, можно сказать, что X = BY, где B = (E - A)-1

B = {bij}n×n, где bij — коэффициент полных материальных затрат — показывает, каким должен быть валовой выпуск i-й отрасли для того чтобы обеспечить выпуск еденицы конечной продукции j-й отрасли.

xij = aij Xj

Взаимосвязь коэффициентов прямых, полных и косвенных затрат

bij включает

  • прямые затраты — израсходовано непосредственно при изготовлении продуката

  • косвеные затраты — относятся к предыдущим этапам произовдства.

B = E + A + A² + A³ + …​ ⇒ (E - A)-1 = B ⇒ bij > aij

Таким образом, коэф. полных затрат не меньше коэф-а прямых затрат, расчитываемого на выпуск 1 валовой продукции j-й отрасли.

Модель межотраслевого баланса затрат труда

При построении межотраслевого баланса затрат труда предполагается, что труд выражается в единицах труда t одинаковой степени сложности — человеко-часах.

\$t_j = L_j / X_j\$, где tj — коэф. прямых затрат труда для производства j-го продукта, Lj — затраты живого труда в производстве продукта, Xi — валовой объём вывпущеной продукции.

\$ ((a_11 T_1, a_12 T_1, ..., a_(1n)T_1), (a_21 T_2, a_22 T_2, ..., a_(2n)T_2), (vdots, vdots, , vdots), (a_(n1) T_n, a_(n2) T_n, ..., a_"nn"T_n), (t_1, t_2, ..., t_n), (T_1, T_2, ..., T_n))\$

Полные затраты труда определяются как сумма прямых затрат живого труда и затрат овеществлённого труда, перенесённых на продукт через израсходованые срадства производства.

\$T_j = sum_(i=1)^n a_(ij)T_j + t_j, j = bar(1,n)\$

где Tj — полные затраты труда на 1 j-го продукта, а \$sum_(i=1)^n a_(ij)T_j\$ — затраты овеществлённого труда, перенесённого на j-й продукт через i-е срадство производства.

Если известны коэф-ы полных материальных затрат bij, можно записать \$T_j = sum_(i=1)^n b_(ij)t_i, j = bar(1,n)\$

T = t B,

где T = (T1, T2, …​, Tn) — коэф-ы полных материальных затрат,

где t = (t1, t2, …​, tn) — коэф-ы прямых материальных затрат.

Планирование потребности в трудовых ресурсах на основе МОБ

Если за год плановый Xj и tj, то общая потребность в затратах труда составит \$L = sum_(j=1)^n t_j X_j\$, где Lдоп — дополнительные затраты труда и затраты в непроизводственной сфере.

Пусть выдвлено m групп трудовых ресурсов и по каждой группе имеются данные о затратах Lkj труда k-й группы в производстве i-го продукта (\$k = bar(1,m)\$), тогда матрица трудовых затрат

\$( (h_11, h_12, ..., h_(1n)), (h_21, h_22, ..., h_(2n)), (vdots, vdots, , vdots), (h_(m1), h_(m2), ..., h_(mn)) )\$

На основании этой матрицы можно получить матрицу коэф-в прямых затрат: \$t_(kj) = h_(kj)/X_j\$.

Эти коэф-ы позволяют рассчитать потребность трудовых ресурсов на плановый объём производства по отдельным группам рабочей силы (профессиям).

  1. с помощью системы уравнений:

\$T_(kj) = sum_(i=1)^n a_(ij)T_(ki) + t_(ki), k = bar(1,m), j = bar(1,n)\$

\$T_(kj) = sum_(i=1)^n b_(ij)t_(ki), k = bar(1,m), j = bar(1,n)\$

Прогнозирование сложных систем

Методологические вопросы прогнозирования сложных систем

Целью прогнозирования является определение характеристик системы в будущем. Это так называемое опережающее отражение. Для возможности этого необходимы определённые предпосылки:

  1. развитие система должно обладать некоторыми закономерностями и устойчивостью связей и отношений;

  2. в развитии системы имеются циклически повторяющиеся ситуации, которые позволяют накопить информацию и реализовать запаздывающее отражение (на основе прошлых данных);

  3. запаздывающее отражение не приводит непосредственно к опережающему отражению, поэтому необходимо наличие активного субъекта, который может оценить опережающий прогноз.

Достоверный прогноз возможен для процессов, которые в значительной степени детерменированы настоящим. Существует две формы детерминации:

  • внутренняя — является более устойчивой, её проще идентифицировать с помощью математических моделей;

  • внешняя — определяется большим числом факторов, которые учесть, как правило, невозможно.

Для прогнозирования необходимо воспроизвести обе формы детерминации.

Метод прогнозирования

совокупность приёмов, позволяющих на основе ретроспективных связей, присущих объекту, а также их изменениях вынести суждение определённой достоверности относительно будущего развития объекта.

Выделяют 3 класса моделей прогнозирования:

  • кривые роста — отражают внутреннюю детерминацию системы;

  • эконометрические модели — оценивают влияние на систему внешних факторов;

  • экспертные оценки — методы на основе интуитивно-логического анализа проблемы с количественной оценой суждений и формальной обработкой результата.

Временны́е ряды и их структура

Временой ряд

последовательность наблюдений некоторого признака случайной велицины Y в последовательные равноотстоящие моменты времени. Отдельные наблюдения — уровни ряда — yt (t = 0, 1, 2, …​, n).

Дискретные/непрерывные, временны́е/интервальные, …​

Иногда для выявления закономерности развития достаточно оценить параметры динамики. Чаще более необходимо подробнее оценить динамику на основе разложения исходного ряда на состовляющие и автономно изучить каждую из них. Это — компонентный анализ. Его основной предпосылкой является утверждение, что реальный процес может быть представлен следующими 4-мя составляющими:

  • ut — тренд, плавно меняющаяся компонента, описывающая влияние долговременных факторов,

  • vt — сезонная компонента, отражающая строго периодическую повторяемость процесса,

  • ct — циклическая компонента, отражающая не строго периодические циклические колебания (циклы Кондратьева (изменения технологических укладов), циклы солнечной активности и т.п.),

  • εt — случайная компонента, отражающая влияние не потдающихся учёту и регистрации случайных факторов.

Используются 2 основные модели:

  • с аддитивной компонентой: yt = ut + vt + ct + εt

  • с мультипликативной компонентой: yt = ut vt ct εt

Классическая задача при исследовании времены́х рядов — оценка основной тенденции.

Основне этапы анализа времены́х рядов:

  1. Графическое представление и описание поведения времено́го ряда

  2. Выделение и удаление закономерных и не случайных состовляющих закономерного ряда (напр. сезонных компонентов)

  3. сглаживание и фильтрация времено́го ряда

  4. построение математической модели для описания тренда

  5. исследование случайной составляющей времено́го ряда и проверка на её основе адекватности математической модели времено́го ряда.

Методы предварительного анализа времены́х рядов

Нехарактерные точки во времены́х рядах делятся на 3 группы:

  1. значения, отражающие объективное развитие процесса, но откланяющиеся от общей тенденции. Не всегда исключаются;

  2. значения, возникающие вследствие изменения методики рассчёта. Точки принимаются за поворотные;

  3. значения, возникающие из-за ошибок. Всегда исключаются;

Методы исключения точек:

  1. нормирование точек:

    1. \$(y_t - bar(Y))/s_y = d*\$, где \$bar(Y)\$ — среднее значение ряда, sy — СКО, d* — пороговое значение для данного ряда. Если неравенство выполняется, то значение считается аномальным, и его заменяют на \$y'_t = alpha_p s_y + bar(Y)\$, и СКО и среднее пересчитываются.

    2. Если наблюдается тренд, то для пересчёта аномальной точки применяется метод Ирвина: \$lambda_t = (y_t - y_(t-1))/s_y\$, которое сравнивается с табличными значениями ́́λ*.

Методы выявляния тренда

Тренд — систематическая составляющая

Метод проверки существенности разности средний

Временной ряд размера n разбивается на 2 части размеров n₁ и n₂.

H₀:\$bar(Y_1) ~= bar(Y_2)\$

Проверка осуществляется в помощью t-критерия Стьюдента. \$t = (bar(Y_1) - bar(Y_2)/s\$, где s — СКО

\$t > t_alpha => H_0\$ отвергается.

Применение этого метода возможно, если временной ряд имеет нормальное распределение и дисперсии половин ряда примерно равны с помощью F критерия Фишера.

Метод Фостера-Стюарда
\$K = sum_(t=1)^b k_t, L = sum_(t=1)^b l_t\$

\$k_t = U_t + V_t\$
\$l_t = U_t - V_t\$
\$U_t = {(1", если " y_t > y_(t-1)"," y_(t-2)"," ..."," y_1), (0):}\$
\$L_t = {(1", если " y_t < y_(t-1)"," y_(t-2)"," ..."," y_1), (0):}\$

L = 0 когда ∑Ut = ∑Vt = 0, то есть тренд отсутствует. Если ∑Ut = ∑Vt ≠ 0, то ряд распадается.

Проверяются гипотезы о том, можно ли считать случайными разности L - μL, μL = 0 и K - μK, где μK — математическое ожидание случайной величины K. Гипотезы проверяются с помощью t-критерия стьюдента (n - 2 степени свободы).

\$t_K = (K-mu_K)/sigma_K\$

\$t_L = L/sigma_L\$

Методы сглаживания временых рядов

Метод простой скользящей средней

Выбирвается длина интервала сглаживания m = 2p+1 и для каждой точке берёся среднее арифметическое её, её p соседей справа и p слева.

\$bar(Y)_t = (sum_(i=t-p)^(t+p) y_i) / m\$

Недостаток — нельзя сгладить первые и последние p членов ряда и они теряются.

Метод экспоненциального сглаживания

Используются только предыдущие значения, взятые с некоторым весом. Сглаженое значение уровня (точки) St определяется по формуле: St = αyt (1 - α)St-1, где α — сглаживающий параметр. Обычно α ∈ [0.1; 0.3], и вес каждого уровня уменьшается экспонициально по мере удаления.

S1 = y1

6. Автокорреляция

Автокорреляция

зависимость значений уровней временого ряда от предыдущих значений того же временого рядя.

Зависимость yt от значений с некоторым запаздыванием во времени. Для получения числовой хараетеристики такой внетреней зависимоти вычисляют кореляционную функцию между исходным рядом yt и этим же рядом, сдвинутым на τ единиц.

Такая функция называется автокореляционной и характеризует внутренюю структуру временого ряда. Состоит из множества коэфицентов автокореляции.

Задавая различные значения τ (τ = 1, 2, …​), получаем последовательные значения коэфицента автокореляции: r1, r2, …​

График автокореляционной функции показывает величину запаздывания, с которым изменение показателей yt сказывается на его последующих значениях.

Величина сдвига τ, которому соответствует наибольший коэфицент автокореляции, называется временым лагом.

7. Общая характеристика и оценка параметров кривых роста

На первом этапе выбирется одна или несколько кривых роста, которые наилучним образом описывают изменение значений.

Полиномы

В полиноме 2 степени первая конечная разность (прирост первого порядка) лиенйно зависит от времени (a1 - a2) + 2 a2t

Экспоненты

yt = (k +) abt

Темп прироста τпр = b - 1 = const

Линеаризация: log yt = log a + t log b, с заменой логарифмов

S-образные кривые

Кривая Гомперца: \$y_t = ka^b^t\$ (k — асимптота)

В ней выделяется 4 участка:

  1. прирост незначителен

  2. прирост увеличивается

  3. прирост постоянен

  4. прирост замедляется

Логарифм данной функции — экспонента.

Логистическая кривая Перла-Рида: \$y_t = k / (1 + ae^(-bt))\$

или \$y_t = k / (1 + 10^(a-bt))\$, \$y_t = k / (1 + ab^(-t))\$

где, а и b — положительные параметры, k — асимптота (предельное значение функции при бесконечном возрастании).

В отличие от гривой Гомперца, имеет точку симетрии, совпадающей с точкой перегиба.

Методы выбора кривой роста

Допустим, есть временой ряд y1, y2, …​, yn. Для выбора вида полиноминальной кривой часто используется метод конечных разностей для предварительного выбора, если уровни временого ряда состоят из 2 компонентов — тренд и случайная компонента. Тренд является достаточно гладким.

На первом этапе вычисляются разности (приросты) до k-го порядко включительно:

  • ut = yt - yt-1

  • ut(2) = ut - ut-1

  • ut(k) = ut(k-1) - ut-1(k-1)

Обычно вычисляют разности до 4 порядка, а далее вычисляют дисперсии для исходного ряда и разностного ряда.

Для исходного ряда:

\$sigma_O^2 = (sum_(t=1)^n y_t^2 - 1/n (sum_(t=1)^n y_t)^2) / (n-1)\$

Для разностного ряда k-го порядка:

\$sigma_k^2 = (sum_(t=k+1)^n (u_t^k)^2) / ((n - k) C_(2k)^k)\$

Далее сравниваются откланения каждой последующей дисперсии от предыдущей: \$|sigma_k^2 - sigma_(k-1)^2|\$, и если эта разность меньше некоторой наперёд заданой величины, то выбирается порядок полинома k-1.

Более универсальным методом является метод характеристик прироста. В соответствии с характером изменения средней прироста и производных показателей, выбирается вид кривой роста с использованием таблицы (см. презентацию).

На практике при предварительном выборе отбирают 2—​3 кривых роста для дальнейшего исследования и построения трендовой модели временого ряда. Для оценки прогнозных свойств модели обычно используется ретроспективный прогноз — во временом ряду, который включает n уровней, выделяют некоторый прогнозный участок n2, который включает n2 последних уровней, а прогноз строится по n1 = n - n2 уровням (то есть, по первым n1 уровням строится прогноз остальных — последних); далее полученые значения сравниваются с фактическими.

Адаптивные модели прогнозирования

В основе экстраполяционных методов прогнозирования предположение о том, что основная тенденция, имевшая место в прошлом, сохранится и в будущем. Если внешние условия меняются, то наиболее важными являются последние реализации процесса, и используют адаптивные модели прогнозирования.

Адаптивные модели быстро бриспосабливают свою структуру и параметры к изменению условий. Наблюдениям присваиваются различные веса в зависимости от того, насколько сильно признаётся их влияние на текущий уровень.

Адаптивные модели используют 2 основные схемы:

  1. Схема скользящего среднего (СС-модели)

  2. Авторегрессии (АР-модели)

В СС-моделях оценкой текущего уровня является взвешеное среднее всех предшествующих уровней. Причём веса убывают по мене удаления от последнего уровня. Задаются параметры зглаживания, значения которых изменяются от 0 до 1. Если параметр больше 0.5, то больший вес имеют последние уровни ряда, если меньше — предшествующие наблюдения имеют больший вес.

В АР-схеме оценкой текущего уровня служит взвешеная сумма не всех, а нескольких предшествующих уровней. Информационная ценность наблюдений определяется теснотой связей между ними, а не их близостью к моделируемому уровню.

Наиболее часто используются 2 базовые СС-модели: Брауна и Хольта. Первая — это частный случай второй. В этих моделях процесс развития представляет собой линейную тенденцию с постоянно изменяющимися параметрами.

Модель Брауна (экспоненциального сглаживания)

Различают 2 модели Брауна:

  • 0-го порядка для описания процессов, не имеющих тенденции. Она имеет одни параметр A0 и прогноз на k шагов вперёд рассчитывается по формуле Y(t+k) = A0

  • 1-го порядка: Y(t+k) = A0 + A1k, где A0 — значение, близкое к последнему уровню, отражает закономерную составляющую, а A1 определяет прирост, сформировавшийся к концу первого периода наблюдений.

  • 2-го порядка отражает развитие в виде параболы: Y(t+k) = A0 + A1k + A2k2

Порядок модели определяют чаще всего визуально.

Этапы построения линейной модели Брауна (1-го порядка):

  1. По первым 5 точкам оцениваются начальные значения A0 и A1 с помощью метода наименьших квадратов для линейной зависимости.

  2. С использованием параметров находим прогноз на 1 шаг вперёд (t = 6).

  3. Определяем ошибку — расхождение прогнозного и фактического значений; если k = l, то e(t+1) = Y(t+1) - -Yp(t, l)

  4. Корректируем параметры: a0(t) = a0(t - 1) + a1(t - 1)k + (1-β2)e(t), a1 = a1(t - 1) + (1 - β)2e(t), где 0 < β < 1 — коэф. дисконтирования данных.

  5. По скорректированным параметрам прогнозируем следующий уровень и повторяем с 3-го пункта

Проверка адекватности моделей прогназирования

Один из основных методов анализа адекватности модели — анализ случайной компоненты et = yt - ut (u — тренд).

Модель является адекватной если значения случайной компоненты являются независимой случайной последовательностью чисел, подчинающейся нормальному закону распределения с постоянной дисперсией.

Анализ адекватности включает проверку

  • равности нулю математического ожидания остаточного ряда,

  • случайности остаточного ряда,

  • независимости остаточного ряда,

  • соответствия значений остаточного ряда нормальному распределению.

Проверка случайности et производится с помощью подсчёта количества пиков p (поворотных точек) — точек, значение которое больше предыдущего и следующего значения или меньше предыдущего и следующего значения.

Должно быть \$p > [(2*(n-2))/3 - 2 sqrt((16n - 29)/90)\$], где n — количество наблюдений (точек), p — количество пиков, [] — целая часть.

Проверка независимости (отсутствия автокореляции). См. методич.

Двойственная задача линейного программирования

Теоремы двойственности:

  1. Выполняется одно из

    1. оптимальные значения прямой и двойственной совпадают

    2. прямая задача не ограничена, а двойственная имеет пустое допустимое множество

    3. двойственная задача не ограничена, а прямая имеет пустое допустимое множество

    4. обе функции имеют пустые допустимые множества

  2. Теорема о дополняемой нежёсткости: решения X (прямой задачи) и Y (двойственной задачи) верные тогда и толко тогда, когда

    1. \$y_i = (sum_(j=1)^n a_(ij) * x_j - b_i) = 0, i = bar(1,m)\$

    2. \$x_i = (sum_(i=1)^n a_(ij) * y_i - c_j) = 0, j = bar(1,n)\$

  3. Теорема об оценках: \$delta f(x) = delta b_i y_i\$
